## What would Earth do?

**Landseed** is a public benefit corporation building the measurement layer for ecological markets.

We deploy [Earth Pulse Nodes](https://landseed.earth) — autonomous sensor arrays that measure ecological condition across six scientific dimensions. The data mints **Earth Credits** — a standardized commodity representing verified ecosystem health.

One acre. One year. Six dimensions. Continuous.

### How it works

```
Land trust partnership
  → Nature Rights Deed recorded
    → Earth Pulse Nodes deployed
      → Ecological condition measured (ECI)
        → Earth Credits minted
          → Credits traded on exchange
            → Buyers retire for TNFD/ESG compliance
```

### The stack

| Layer | What |
|-------|------|
| **Measurement** | Earth Pulse Nodes — acoustic, soil, weather, camera, water |
| **Science** | SEEA EA framework — six dimensions, penalized geometric mean |
| **Legal** | Nature Rights Deed — 100-year term, county-recorded |
| **Registry** | Earth Credit minting, tracking, retirement |
| **Market** | Exchange, prediction markets, Earth Signals |
